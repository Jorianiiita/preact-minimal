# :rocket: preact-seed
Minimal preact structure

### Stack

* preact
* preact-router
* babel
* webpack 2

### Usage

Clone the project 

```bash
git clone git@github.com:aganglada/preact-seed.git
```

Enter the folder

```bash
cd preact-seed 
```

Install dependencies

```bash
yarn 
```

If you are using npm...

```bash
npm i
```

Run the dev server

```bash
npm run dev
```

You can start developing now on http://localhost:4000
