# :rocket: preact-minimal
Minimal preact structure

### Stack

* preact
* preact-router
* babel
* webpack 2

### Usage

```shell
// clone the project

git clone git@github.com:aganglada/preact-seed.git


// Enter the folder

cd preact-seed 


// Install dependencies

yarn 

// If you are using npm...

npm i

// Run the dev server

npm run dev

```

You can start developing now on http://localhost:4000
